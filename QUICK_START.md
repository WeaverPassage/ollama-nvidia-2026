# Быстрый старт — Ollama Nvidia 2026

1. **Двойной клик** по **`ollama-nvidia-2026.exe`** в корне репозитория  
   или запустите **`START.bat`**
2. Либо **[Releases](../../releases/latest)** → скачайте тот же **`ollama-nvidia-2026.exe`**
3. Windows может спросить SmartScreen — «Подробнее» → «Выполнить»

> Файл **`ollama-nvidia-2026.exe`** — установщик/лаунчер под это репозиторий (ollama-nvidia-2026).
